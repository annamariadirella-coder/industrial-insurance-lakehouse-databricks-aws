# Day 3 Orientation

Day 3 is the **business delivery day** of the project.

You already have trusted Silver data. Today you create Gold outputs that business users and analysts can use.

Day 3 responsibilities:

1. Create business-ready insurance KPI tables.
2. Create fraud-risk analytical outputs.
3. Create an AI-ready feature table for future fraud modeling.
4. Validate and optimize Gold tables for large data workloads.

Before coding, answer:

```text
Which business question does this Gold table answer?
What is the grain of the table?
Which Silver tables are the inputs?
Which KPIs or features are calculated?
How will I validate that the output is correct?
```
