# Insurance KPI Logic

| KPI | Description |
|---|---|
| total_customers | Number of customers |
| total_policies | Number of policies |
| active_policies | Policies with status active |
| premium_revenue | Sum of premium amount |
| total_claims | Number of claims |
| open_claims | Claims with status open |
| approved_claims | Claims with status approved |
| rejected_claims | Claims with status rejected |
| paid_claims | Claims with status paid |
| total_claim_amount | Sum of claim amount |
| average_claim_amount | Average claim amount |
| total_paid_amount | Sum of paid payments |
| payment_rejection_rate | Rejected payments / total payments |
| claims_ratio | Total claim amount / premium revenue |
| fraud_risk_rate | High-risk claims / total claims |
| average_risk_score | Average fraud risk score |
| average_payment_delay_days | Average delay between claim and payment |
| estimated_commission | Premium revenue * commission rate |

Always document the formula you use.
