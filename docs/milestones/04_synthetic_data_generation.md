# Milestone — Synthetic Data Generation


Generate realistic German insurance data using PySpark.

Required datasets:
- customers
- policies
- claims
- payments
- agents
- fraud_indicators

Start with `DATA_MODE = "small"`.

Include:
- German cities
- Bundesländer
- German-style postal codes
- GDPR consent
- insurance product types
- SEPA/bank-transfer style fields
- synthetic-only personal data
