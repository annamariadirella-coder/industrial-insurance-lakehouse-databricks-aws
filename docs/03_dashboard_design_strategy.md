# Dashboard Design Strategy

Dashboard design starts with business questions.

For each dashboard page, define:

1. Who is the audience?
2. What decision or monitoring question does the page support?
3. Which Gold table or view is the source?
4. Which KPIs should appear?
5. Which dimensions should be filterable?
6. Which fields must not be exposed because of PII/GDPR concerns?

Required dashboard pages:

1. Executive Insurance Overview
2. Claims Operations
3. Policy Portfolio
4. Fraud Risk Monitoring
5. Agent and Regional Performance
6. Data Quality Monitoring
