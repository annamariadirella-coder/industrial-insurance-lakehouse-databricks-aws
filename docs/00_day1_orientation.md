# Day 1 Orientation

Day 1 is the **industrial data foundation day**.

You are not building Silver data quality rules, Gold KPIs, dashboards, or AI features yet.

Today you build:

```text
Trial workspace → AWS S3 bucket → synthetic data → raw landing → Bronze Delta tables → validation
```

Think like a cloud data engineer working for a German insurance company. Your first responsibility is to create a safe, organized, traceable, and scalable foundation.
