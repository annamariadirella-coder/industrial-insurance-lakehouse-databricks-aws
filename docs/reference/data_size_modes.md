# Data Size Modes

| Mode | Customers | Policies | Claims | Payments |
|---|---:|---:|---:|---:|
| small | 10,000 | 25,000 | 50,000 | 50,000 |
| medium | 500,000 | 1,500,000 | 5,000,000 | 5,000,000 |
| large | 2,000,000+ | 5,000,000+ | 20,000,000+ | 20,000,000 |

Recommendation:
1. Run small mode.
2. Validate the pipeline.
3. Try medium if compute and cost allow.
4. Treat large as optional or instructor-led.
