# Day 1 Learning Goals

By the end of Day 1, you should understand:

- how the Databricks trial differs from Free Edition
- how AWS S3 acts as external cloud lakehouse storage
- how to organize S3 folders for raw, bronze, checkpoints, and quarantine
- how to generate large synthetic insurance datasets with Spark
- why raw files should land in S3 before Bronze ingestion
- how Bronze Delta tables preserve ingestion traceability
- why ingestion metadata matters
- how to validate raw-to-Bronze row counts
- why cost control matters when using real cloud resources
